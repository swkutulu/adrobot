1. Забрать из git и перейти в папку
2. Подготовить виртуальное окружение.
Например:
> uv sync
Можно любым другим способом, главное что бы в окружении был requests
3. Запустить с параметрами по умолчанию
> python main.py
Будет сделано 10 запросов на адрес https://speedtest.selectel.ru/10MB
4. Параметры можно прменять
> python main.py --help
